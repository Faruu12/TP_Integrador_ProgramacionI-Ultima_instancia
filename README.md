# TP_Integrador_ProgramacionI-Ultima_instancia
Este trabajo fue realizado por Axel Ponce
Este proyecto consiste en un sistema desarrollado en Python que permite gestionar información de países utilizando estructuras básicas de programación.
Incluye carga y guardado de datos en un archivo CSV, búsquedas, filtros, ordenamientos y estadísticas.

El objetivo principal es integrar todos los contenidos trabajados en la materia: listas, diccionarios, funciones, archivos, ciclos, condicionales y modularización.
